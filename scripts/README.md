# Скрипты для оптимизации картинок

## Конвертация картинок Prime в WebP

В этой папке находятся два скрипта для конвертации картинок вкусов Prime в формат WebP для улучшения производительности сайта.

### Скрипты:

#### 1. `convert-prime-to-webp.js`
Конвертирует все PNG картинки Prime во WebP формат.

**Использование:**
```bash
node scripts/convert-prime-to-webp.js
```

**Что делает:**
- Проходит по всем картинкам Prime (находятся в папке `img/img-card-prime-*.png`)
- Конвертирует каждую в WebP формат с качеством 80
- Сохраняет WebP файлы рядом с оригиналами
- Выводит прогресс конвертации

#### 2. `update-html-to-webp.js`
Обновляет HTML файл, чтобы использовать новые WebP картинки вместо PNG.

**Использование:**
```bash
node scripts/update-html-to-webp.js
```

**Что делает:**
- Заменяет все ссылки на PNG картинки Prime в `index.html` на WebP
- Сохраняет обновленный HTML файл
- Выводит количество замененных ссылок

### Рекомендуемый порядок:

1. Сначала установите зависимость `sharp`:
```bash
npm install sharp
```

2. Запустите скрипт конвертации:
```bash
node scripts/convert-prime-to-webp.js
```

3. Затем обновите HTML:
```bash
node scripts/update-html-to-webp.js
```

4. Проверьте результаты в браузере

### Преимущества WebP:

- ✅ Меньший размер файлов (обычно на 25-30% меньше, чем PNG)
- ✅ Быстрая загрузка страницы
- ✅ Лучшая производительность
- ✅ Поддерживается всеми современными браузерами

### Откат (если необходимо):

Если нужно вернуться к PNG картинкам, просто:
1. Запустите `git checkout index.html` (если используется git)
2. Удалите WebP файлы из папки `img/`
