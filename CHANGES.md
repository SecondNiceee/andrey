## 2️⃣ Оптимизация картинок Prime (конвертация в WebP)

✅ **Статус:** Скрипты готовы к использованию

Созданы два Node.js скрипта для автоматической конвертации картинок вкусов Prime в формат WebP:

### Созданные файлы:

1. **`scripts/convert-prime-to-webp.js`**
   - Конвертирует все 56 PNG картинок Prime во WebP
   - Использует библиотеку `sharp`
   - Сохраняет файлы с качеством 80%

2. **`scripts/update-html-to-webp.js`**
   - Обновляет `index.html` для использования WebP картинок
   - Заменяет все ссылки автоматически
   - Сохраняет резервную копию в памяти

3. **`package.json`**
   - Добавлена зависимость `sharp`
   - Добавлены удобные npm скрипты

4. **`scripts/README.md`**
   - Подробная инструкция по использованию скриптов

### Как использовать:

```bash
# 1. Установите зависимости
npm install

# 2. Вариант 1: Запустить все за раз
npm run optimize-images

# ИЛИ Вариант 2: Запустить отдельно
npm run convert-prime-webp      # Конвертирует картинки
npm run update-html-webp        # Обновляет HTML
```

### Преимущества:

- ✅ WebP картинки на 25-30% меньше чем PNG
- ✅ Быстрая загрузка страницы
- ✅ Поддерживается всеми современными браузерами
- ✅ Автоматическое обновление HTML
